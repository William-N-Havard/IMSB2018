Question 2

La tartine n'ayant que deux faces, le rsultat s'apparente un tirage sur une loi binomiale de paramtre \( \theta \) inconnu. Quelle est la distribution posterieure du paramtre \( \theta \) associe ces donnes sachant que l'analyste n'avait aucun a priori.

Rponse 2

La tartine n'ayant que deux faces, le rsultat s'apparente un tirage sur une loi binomiale de paramtre \( \theta \) inconnu. Quelle est la distribution posterieure du paramtre \( \theta \) associe ces donnes sachant que l'analyste n'avait aucun a priori. 

nbTrial <- length(data$trial)
nbSucces <- sum(data$value)
grid_size <- 1e3
p_grid     <- seq(from = 0., to = 1., length.out = grid_size)
prior      <- rep(1, grid_size)
likelihood <- dbinom(nbSucces, size = nbTrial, prob = p_grid)
posterior  <- likelihood * prior
posterior  <- posterior / sum(posterior)
str(posterior)
